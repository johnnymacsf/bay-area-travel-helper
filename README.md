# Bay Area Travel Helper

A helpful way to discover and review hotels in the San Francisco Bay Area! Search for hotels, read user reviews, add your own review, and get AI-generated summaries for a hotel.

## Features
- Search for hotels by keyword in the hotel name
- Paginated results for hotels and reviews
- AI-generated pros/cons list summarizing the hotel's reviews
- Create your own reviews
- Direct links to Expedia hotel listing
- User authentication (login and sign up)
- View hotel and review details

## Tech Stack
- **Frontend:** React 19, Tailwind CSS
- **Backend:** Spring Boot 4.0.1 (Java 21), MySQL, Spring Data JPA
- **AI:** LangChain4j, OpenAI
